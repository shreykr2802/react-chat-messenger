# React Chat Messenger

Simple React Application for realtime chat.

## Description

This project is a simple yet powerful real-time chat application developed using React. The application allows users to engage in seamless conversations with one another, with the capabilities of React and Pocketbase.

## Getting Started

### Dependencies

- React
- Poacketbase

### Installing and Running

- `git clone https://github.com/shreykr2802/react-chat-messenger.git`
- `bun install`
- `bun dev`
